Karvinen 2022: Cracking Passwords with Hashcat
https://terokarvinen.com/2022/cracking-passwords-with-hashcat/

- hashcatin tarvitsee tietää murrettavan hash-algoritmin tyyppi onnistuakseen, kannattanee vertailla hash-algoritmeja 'hashcat --example-hashes'
- parametri -m on hashmode eli sen "tyyppi" ID
- artikelissa kerrotaan hyvin hascatin eri tulosteiden merkitys


Karvinen 2023: Crack File Password With John
https://terokarvinen.com/2023/crack-file-password-with-john/

- ladataan ennemmin Johin jumbo versio, sillä se tukee huomattavasti enemmän hash tyyppejä
- ei kannata käyttää apt installia, koska se voi asentaa vanhemman version ja ilman jumbo lisää
- ./configure tarkistaa järjestelmän ja etsii kirjastot
- parametrilla -j4 voidaan käyttää useampaa CPU ydintä
