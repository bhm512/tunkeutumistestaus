# h4 Täysin Laillinen Sertifikaatti

### x)

OWASP 2021: OWASP Top 10:2021

- access control pitää huolen siitä, että käyttäjillä on pääsy vain niihin tietoihin/resursseihin joihin tällä on oikeus

  Yleisimmät haavoittuvuudet:
  - liian laajat oikeudet
  - pääsynhallinan ohittaminen
  - IDOR (pääsy toisen tietoihin pelkällä tunnisteella)
  - oikeuksien korostus

  Miten estää?
  - deny by defalut oletukseksi
  - pääsynhallintaa voi tehdä myös palvelin puolella
  - katsoa että käyttäjä voi käyttää vain omia tietojaan
  - käytä lokia ja valvo sitä epäonnistuneiden yritysten kohdalla
  - rate limiting

IDOR = haavoittuvuus, jossa sovellus käyttää käyttäjän tunnistetta tietojen hakemiseen, ILMAN kunnollista käyttöoikeus tarkistusta.

Path Traversal = haavoittuvuus, jossa hyökkääjä voio päästä käsiksi arkaluontisiin materiaaleihin manipuloimalla hakemistopolkua.

Cross-site scripting = haavoittuvuus, jossa hyökkääjä pääsee ajamaan omaa JavaScript koodia toisen käyttjän selaimessa


### a) Totally Legit Sertificate. Asenna OWASP ZAP, generoi CA-sertifikaatti ja asenna se selaimeesi. Laita ZAP proxyksi selaimeesi. Laita ZAP sieppaamaan myös kuvat, niitä tarvitaan tämän kerran kotitehtävissä. Osoita, että hakupyynnöt ilmestyvät ZAP:n käyttöliittymään. (Voi vaatia Firefox about:config network.proxy.allow_hijacking_localhost. Foxyproxy laittoi tämän aiemmin päälle itse. Kalin Firefox ESR oli viimeksi ongelmia Foxyproxyn kanssa - vaihtoehtona on asettaa Proxy käsin Settings, hakusana "proxy")

Asensin ja käynnistin zaprxoyn 

```
sudo apt install zaproxy
zaproxy
```
