# h5 Fuzzy

x) Karvinen 2023: Find Hidden Web Directories - Fuzz URLs with ffuf

- Fuzzing = koitetaan löytää haavoittuvuuksia verkkosovelluksista syöttämällä niille läjäpäin odottamatonta dataa.
- Fuff on fuzzing työkalu, jolla voi löytää esim piilotettuja hakemistoja. Sitä voi tosin soveltaa myös esim otsikoiden tai muiden parametrien fuzzaamiseen.
- kerrottiin miten fuffin avulla fuzzaaminen on nopeampaan, kuin manuaalisesti.

  https://github.com/ffuf/ffuf/blob/master/README.md

  - joohoin (Joona Hoikkalan) hyvät ohjeet asennukseen ja peruskäyttöön liittyen
  - Hyvät videot auttavat ymmärtämään käytäntöä


### a) Fuzzzz. Ratkaise dirfuz-1 artikkelista Karvinen 2023: Find Hidden Web Directories - Fuzz URLs with ffuf.

Asensin alkuun ffufin valmiina pakettina 

```
$ sudo apt-get update
$ sudo apt-get install ffuf
```

Sen jälkeen latasin kansion/listan, josta ffuf ottaa kohteeseen testattavat asiat. Latasin sen myös Teron sivuilta komennolla

```
wget https://raw.githubusercontent.com/danielmiessler/SecLists/master/Discovery/Web-Content/common.txt
```
